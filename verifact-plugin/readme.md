# Verifact — WordPress Plugin (2.0.5)
UI + REST proxy for the VeriFact FastAPI service.

## Install
- Copy this verifact-plugin folder to wp-content/plugins/
- Activate in Admin → Plugins
- Configure Settings → VeriFact (API base, rate limit, cache, sources)

## Features
- Shortcode UI with claim or Prompt+Answer
- REST route verifact/v1/check proxy to FastAPI /check
- Admin dashboard, analytics, history & logs, bulk tools, API management, settings, user management
- Database logging table wp_verifact_logs
- Rate limiting (with role overrides) and transient/object caching
- WP-Cron scheduled checks
- Optional sources: Archive.org, Grokopedia
- Remote cache upload via S3 presigned URL
